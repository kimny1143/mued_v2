# MUEDnote PoC - 質問品質評価シート

## 評価シート使用方法

このシートは、MUEDnoteが生成した質問の品質を評価するためのテンプレートです。評価者（講師・プロ作曲家・プロジェクトオーナー）が使用します。

### 評価手順

1. テスター（ユーザー）から実際のSession情報を受け取る
2. MUEDnoteが生成した質問を確認する
3. 各質問を3つの評価軸で採点する（1-5点）
4. 総合評価とコメントを記入する

## 評価項目（各1-5点）

### 評価軸の定義

#### 1. 関連度（Relevance）
ユーザーが**今やっている作業**に対して、質問が適切に刺さっているか。

- **5点**: userShortNoteの内容にピンポイントで刺さっている。聞かれて当然の質問。
- **4点**: 今の作業に関連しているが、やや広範囲または少しずれている。
- **3点**: 一般的な質問で、今の作業に特化していない。
- **2点**: 今の作業とはあまり関係ない。
- **1点**: 完全にズレている。聞かれても困る。

#### 2. 筆のノリ（Answerability）
ユーザーが**答えやすいか**。答えることで頭が整理されるか。

- **5点**: すぐに答えられそう。答えたら作業が進みそう。
- **4点**: 少し考えれば答えられる。悪くない質問。
- **3点**: 答えられるが、やや抽象的または堅苦しい。
- **2点**: 答えにくい。理論的すぎる／抽象的すぎる。
- **1点**: 答えたくない。プレッシャーを感じる。

#### 3. 深度（Depth）
質問の**深さ**が適切か。浅すぎず、重すぎず。

- **5点**: ちょうどいい深さ。初心者でもプロでも考える価値がある。
- **4点**: やや浅いまたは深いが、許容範囲。
- **3点**: 浅すぎる（当たり前のこと）または深すぎる（哲学的）。
- **2点**: 明らかに浅すぎる／深すぎる。
- **1点**: 質問として成立していない。

### 合否判定

- [ ] **合格**: 平均スコア 3.5点以上、かつ「完全にズレている質問」が0-1個
- [ ] **要改善**: 平均スコア 2.5-3.4点、または「完全にズレている質問」が2個
- [ ] **不合格**: 平均スコア 2.4点以下、または「完全にズレている質問」が3個

---

### 完全にズレている質問の有無

- [ ] なし（全て許容範囲内）
- [ ] 1個あり：質問［  ］番
- [ ] 2個あり：質問［  ］番、［  ］番
- [ ] 3個あり（全てズレている）

---

### 改善提案

**良かった点**
```
［例：質問1は、userShortNoteの内容に完全に一致していて、答えやすかった］
```

**改善すべき点**
```
［例：質問2が抽象的すぎる。もっと具体的な楽曲例やシチュエーションを提示した方が良い］
```

**テンプレートへの提案**
```
［例：「Mixing系のSessionでは、周波数帯域に関する質問を必ず1つ入れる」というルールがあると良い］
```

## 付録：評価例（参考）

### 評価例1：高品質な質問

**Session情報**
- **Type**: new_composition
- **Title**: 「バラード曲のサビ制作」
- **userShortNote**: 「サビのコード進行がワンパターンで、感動が薄い。もっとドラマチックにしたい。」

**生成された質問**
```
今回のコード進行で、「もっとドラマチックに」したいと思った具体的な箇所は、
サビの何小節目あたりですか？また、どんな感情を表現したかったですか？
```

**評価**
- **関連度**: 5点（userShortNoteの「ドラマチック」に直結）
- **筆のノリ**: 5点（具体的で答えやすい。小節数と感情という2つの観点が明確）
- **深度**: 5点（浅すぎず深すぎず。プロでも初心者でも答えられる）

---

### 評価例2：改善が必要な質問

**Session情報**
- **Type**: mixing
- **Title**: 「ボーカルミックス最終調整」
- **userShortNote**: 「ボーカルがオケに埋もれる。EQとコンプで調整中。」

**生成された質問**
```
音楽制作において、あなたが最も大切にしている価値観は何ですか？
```

**評価**
- **関連度**: 1点（完全にズレている。今のミックス作業と全く関係ない）
- **筆のノリ**: 2点（哲学的すぎて答えにくい）
- **深度**: 2点（深すぎる。今の作業に必要ない）

**改善提案**: 「ボーカルのEQ調整で、どの周波数帯域をカット／ブーストしましたか？」など、具体的な技術質問にすべき。
