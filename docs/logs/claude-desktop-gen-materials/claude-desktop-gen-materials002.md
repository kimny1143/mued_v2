## 1. 構造設計

- **総小節数**: 32小節（上級者レベル）
- **形式**: A(8小節) - B(8小節) - A'(8小節) - Coda(8小節)
- **調性**: D minor → F major（相対長調への転調で希望と緊張を表現）
- **テンポ**: 85 BPM（劇伴の感情的なテンポ）
- **拍子**: 4/4

## 2. 教育的統合

**コア技術スキル**:
1. ストリングスセクションのレイヤリング（5パート: Vln I, Vln II, Vla, Vc, Db）
2. ベロシティカーブによる表現力のあるフレージング
3. 和声的ヴォイシングとディヴィジ（divisi）テクニック

**旋律的興味**: 
- セクションA: 憂鬱で内省的な旋律
- セクションB: 緊張感の高まり、クライマックスへ
- セクションA': 変奏された再現
- Coda: 希望へ向かう解決

## 3. ストリングスアレンジの難易度設定

**intermediate（中級者）向け**:
- **音域**: 各楽器の実用的な中音域〜高音域
- **リズム**: 8分音符、付点音符、シンコペーション、トレモロ効果
- **和声**: 豊かな和音、時折のディヴィジ
- **奏法**: レガート、スタッカート、ピチカート、スル・ポンティチェロ

## 🎼 補足: ProToolsでの実践ポイント

### 推奨音源ライブラリ
- **Spitfire Audio**: Albion One, BBC Symphony Orchestra
- **EastWest**: Hollywood Strings
- **Native Instruments**: Kontakt + Session Strings

### MIDI CC活用
- **CC1 (Modulation)**: ビブラートの深さ
- **CC11 (Expression)**: 全体的な音量/表現力
- **CC64 (Sustain)**: レガート奏法の補助

### 実践的なヒント
1. 各パートを別々にソロで確認
2. ベロシティのヒューマナイズ機能を活用(±5-10)
3. クオンタイズは100%ではなく85-90%に設定して人間的なタイム感を残す
