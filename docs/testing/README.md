# テスト

## クイックコマンド

```bash
npm run test           # ユニットテスト (Vitest)
npm run test:e2e       # E2E テスト (Playwright)
npm run typecheck      # 型チェック
npm run lint           # Lint
```

## ドキュメント

- [TESTING_GUIDE.md](TESTING_GUIDE.md) - テスト戦略・実行方法の詳細

## MCP サーバー

Claude Desktop から直接テスト実行可能：
- `mued_unit_test` - ユニットテスト
- `mued_e2e` - E2E テスト
